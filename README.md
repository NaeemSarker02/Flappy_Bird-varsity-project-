# Flappy Bird Web Game

Simple browser-based Flappy Bird game built with HTML, CSS, and JavaScript.

## Free Deployment

This repository is configured for free deployment on GitHub Pages.

1. Push your latest code to `master`.
2. Open the GitHub repository `NaeemSarker02/Flappy_Bird-varsity-project-`.
3. Go to `Settings` -> `Pages`.
4. Under `Build and deployment`, set `Source` to `GitHub Actions` if it is not already selected.
5. After the workflow finishes, the game will be available at:

`https://naeemsarker02.github.io/Flappy_Bird-varsity-project-/`

If the site shows a 404 at first, wait 1 to 2 minutes and refresh after the Pages deployment completes.
